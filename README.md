# Vishal Flore Weddings | Luxury Floral Decors & Event Planning

A premium, production-ready web application for a luxury wedding decoration and event planning service. This project features high-quality aesthetics, smooth animations, and a responsive design tailored for a professional client-facing presence.

- **Premium Design Aesthetics**: Luxury-themed UI with a curated color palette (Gold, Ivory, Burgundy) and modern typography.

- **Service Portfolio**: Detailed service descriptions for Wedding Decoration, Floral Artistry, DJ Services, and Event Management.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop experiences.
- **Contact & Inquiries**: Functional contact form (simulated) and direct WhatsApp/Phone integration.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm or bun

### Installation

1. Clone the repository:

   ```sh
   git clone <repository-url>
   cd vishal-flore-weddings
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start the development server:

   ```sh
   npm run dev
   ```

4. Build for production:
   ```sh
   npm run build
   ```

## üì¶ Deployment on Vercel

This project is optimized for deployment on Vercel.

1. Push your code to a GitHub/GitLab/Bitbucket repository.
2. Import the project in Vercel.
3. The build settings should automatically be detected:
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
   - **Framework Preset**: `Vite`

### Environment Variables

If you decide to connect a real backend, you can add the following environment variable in Vercel:

- `VITE_API_URL`: Your backend API endpoint.
